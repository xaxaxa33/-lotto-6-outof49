# Lotto 6 out of 49

Simple JavaScript application that allows users to select 6 numbers and check them against randomly drawn lottery numbers.

## Description
This project allows users to:
- select 6 numbers between 1 and 49
- generate 6 random lottery numbers
- see how many numbers match
- view results in a clear, interactive format

The application includes input validation to ensure numbers are within range and unique. Core logic is separated into modular, testable JavaScript functions for easy maintenance and extension.

## Technologies
- HTML
- CSS
- JavaScript (ES modules)
- DOM manipulation for interactive UI
